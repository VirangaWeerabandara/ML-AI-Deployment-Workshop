🌐

Real-time Inference

Model wrapped in an API. User sends request → model infers → result returned immediately.

Example: ChatGPT, fraud detection
📅

Batch Inference

Predictions generated on a schedule for large datasets, stored in a database.

Example: Netflix recommendations overnight
📱

Edge Deployment

Model runs directly on user's device — no cloud needed. Great for privacy & offline.

Example: On-device face recognition
⏱️

Latency

Time to process one request. Measured in ms. Critical for real-time UX.

Target: <100ms for APIs, <1s for LLMs
📈

Throughput

Requests handled per second. Critical for scaling to millions of users.

Trade-off: Batching ↑ throughput but ↑ latency
📐

Quantization

Reducing weight precision (FP16→INT4). Cuts memory 4x with minimal accuracy loss.

7B model: 14GB → 3.5GB VRAM
🐳

Docker

Packages OS + dependencies + code into a portable container. Eliminates "works on my machine."

Build once, deploy anywhere
⚡

vLLM & PagedAttention

Manages LLM memory like OS virtual memory. Batches requests for 10-20x throughput.

Industry standard for production LLMs
